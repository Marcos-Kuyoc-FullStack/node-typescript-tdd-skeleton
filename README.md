# API Skeleton usando Node y TypeScript
Bienvenido a nuestra API Base escrita en Node y Typescript para el manejo de nuestros proyectos, puedes clonar y comenzar a trabajar en tus propios desarollos

## Installation

```bash
$ yarn install
```

## Running the app
```bash
# development
$ docker-compose up -d
```

```bash
# development
$ yarn run start

# watch mode
$ yarn run develop

# production mode
$ npm run start
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
